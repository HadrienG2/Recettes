# Livre de recettes commun

Ce dépôt contient les sources du livre de recettes qui est en ligne à l'adresse
<https://hadrieng2.github.io/recettes/>.

Votre mission si vous l'acceptez est de proposer des changements ou ajouter
d'autres recettes via une _pull request_. Il faut pour cela créer un _fork_,
puis y pousser une branche qui modifie un fichier existant dans le dossier
`recettes`, ou bien qui crée un nouveau fichier en vous inspirant du fichier
[`recettes/crepes.md`](recettes/crepes.md?plain=1) fourni.

Retrouvez plus d'informations sur ce processus dans [les supports du TP associé
à ce
dépôt](https://grasland.pages.in2p3.fr/formation-git/part2/github/3-contribution.html).
